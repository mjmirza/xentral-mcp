Self-hosting question. Modern Xentral is cloud-only and cannot be self-hosted.
The only self-hostable option is the legacy OpenXE fork, which this MCP cannot
talk to. The full write-up is in the separate
[xentral-self-hosting-lessons](https://github.com/mjmirza/xentral-self-hosting-lessons)
repository.
